# U-Net 音频降噪器项目概述

## 1. 项目简介

这个项目提出了一种基于U-Net架构的新方法来解决音频降噪问题。U-Net原本是用于图像分割的深度学习模型,在这里被创新性地改编用于音频降噪任务。

## 2. 数据创建

- **数据来源**:
  - 干净语音: LibriSpeech和SiSec数据集
  - 环境噪声: ESC-50数据集(10类噪声)
- **数据处理**:
  - 采样率: 8kHz
  - 窗口长度: 约1秒
  - 噪声增强: 对原始噪声进行数据增强
  - 混合比例: 20%-80%的随机噪声水平与干净语音混合
- **数据集规模**:
  - 训练数据: 10小时的带噪语音和对应的干净语音
  - 验证数据: 1小时

## 3. 模型训练

- **模型架构**: 改编的U-Net模型
- **输入**: 带噪语音的幅度谱
- **输出**: 噪声模型(带噪语音谱 - 干净语音谱)
- **网络结构**:
  - 编码器: 10个卷积层
  - 解码器: 对称结构,带跳跃连接
- **训练细节**:
  - 优化器: Adam
  - 损失函数: Huber损失
- **训练结果**:
  - 训练损失: 0.002129
  - 验证损失: 0.002406

## 4. 预测流程

1. 将带噪音频转换为时间序列窗口
2. 将窗口转换为幅度谱和相位谱
3. U-Net预测每个窗口的噪声模型
4. 从带噪谱中减去噪声模型
5. 将处理后的幅度谱与原始相位结合
6. 进行ISTFT(逆短时傅里叶变换)得到降噪后的时间序列

## 5. 用户界面

- **技术**: Streamlit Web应用
- **功能**:
  - 音频文件上传
  - 使用预训练U-Net模型进行降噪
  - 可视化输入输出音频
  - 展示各种音频分析结果

## 总结

这个项目展示了如何将深度学习技术应用于音频处理领域,特别是降噪任务。它提供了从数据准备到模型训练再到实际应用的完整流程,为音频处理领域的研究者和开发者提供了宝贵的参考。
